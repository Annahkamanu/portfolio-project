Weather app

For my second developer project, I aimed to create a weather web app by integrating the OpenWeather API. This app allows users to easily check the current weather for any city worldwide. Additionally, it provides forecasts with appropriate icons for each weather condition. This project was both thrilling to work on and served as a test of my JavaScript skills and comprehension.
